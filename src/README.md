# Source

Application source for **Meccha Chameleon Game Trainer** — companion overlay for [MECCHA CHAMELEON](https://store.steampowered.com/app/4704690/MECCHA_CHAMELEON/).

Planned modules:

- `Core/` — window attach, overlay rendering (DirectX hook)
- `Profiles/` — paint preset and pose JSON loader
- `Overlay/` — timer, player roster, hide-spot markers
- `Input/` — global hotkey binding

Build instructions are added when the binary pipeline is wired up.
